## This site markets itself

The portfolio practices what it preaches:

- **Open Graph + Twitter cards** with a branded 1200×630 share image — links look polished everywhere
- **JSON-LD structured data** (`Person` schema) for rich search results
- **Sitemap + robots.txt** for crawlability
- **GA4 event tracking** on every contact touchpoint (`contact_click` with channel + placement) and SPA page-view tracking across routes
- **Per-route titles and meta descriptions**
- **Conversion-focused UX** — availability badge, hero CTAs, proof stats, and a closing contact section
